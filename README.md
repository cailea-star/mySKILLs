# M2 Agents

Minimal backup repo for Codex skills.

## Skills

- `introduction-writing`: reference-grounded introductions and opening reports.
- `numerical-utility-module-development`: C++ numerical/scientific utility modules.
- `reference-code-reproduction`: reproduce trusted scientific code with diagnostic tests.
- `scientific-code-review-and-minimal-fix`: review/fix scientific code with smallest safe change.
- `theoretical-correctness-audit`: check theory/model/numerics before implementation/debugging.

## Layout

- `skills/<name>/SKILL.md`: skill instructions.
- `skills/<name>/agents/openai.yaml`: optional app metadata.
- `obsolete/`: retired agent prompts.
